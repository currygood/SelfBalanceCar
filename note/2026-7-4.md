## 角度环使用GyroZ作为Actual

// 注意：GyroZ 也要减去零漂补偿，类似 GyroX
float current_yaw_rate = (float)mpu6050Data.GyroZ / SENSITIVITY; 

// 2. 将转向环的输入改为角速度
TurnPID.Actual = current_yaw_rate; 

PID_Update(&TurnPID, 0);
DifPWM = TurnPID.Out;

为什么要使用GyroZ作为Actual?
- 响应快
- 因为GyroZ是小车的偏航角,而小车的偏航角是需要控制的,所以使用GyroZ作为Actual.
