### **引脚分配表**

| 功能               | 引脚     | 定时器/外设      |
| ------------------ | -------- | ---------------- |
| **编码器1 A相**    | **PA6**  | TIM3_CH1         |
| **编码器1 B相**    | **PA7**  | TIM3_CH2         |
| **编码器2 A相**    | **PB6**  | TIM4_CH1         |
| **编码器2 B相**    | **PB7**  | TIM4_CH2         |
| **I2C SCL**        | **PB8**  | GPIO (软件I2C)   |
| **I2C SDA**        | **PB9**  | GPIO (软件I2C)   |
| **蓝牙 USART1_TX** | **PA9**  | USART1           |
| **蓝牙 USART1_RX** | **PA10** | USART1           |
| **超声波 Trig**    | **PA4**  | GPIO_Output      |
| **超声波 Echo**    | **PA5**  | GPIO_EXTI 或 TIM |
| **PWMA（左电机）** | **PA8**  | TIM1_CH1         |
| **PWMB（右电机）** | **PA11** | TIM1_CH4         |
| **AIN1**           | **PB12** | GPIO_Output      |
| **AIN2**           | **PB13** | GPIO_Output      |
| **BIN1**           | **PB14** | GPIO_Output      |
| **BIN2**           | **PB15** | GPIO_Output      |
